## ğŸ“Š Project Summary

### What is ATS Scanner 2.0?
An AI-powered web application that analyzes resumes against job descriptions using Google's Gemini LLM to help job seekers optimize their resumes for Applicant Tracking Systems (ATS).

### Key Statistics:
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Python 3.12, FastAPI, Uvicorn
- **Database:** MongoDB Atlas (Cloud)
- **API Endpoints:** 14+ RESTful endpoints
- **Collections:** 2 (users, scans)
- **External Services:** 2 (Google Gemini, Gmail SMTP)
- **Deployment:** Render (backend), Vercel (frontend)

### User Roles:
1. **Guest User** - Analyze without account
2. **Registered User** - Save scan history
3. **Verified User** - Full access with email verification

### 4 Core Functions:
1. **Resume Analysis Engine** - AI-powered analysis
2. **User Authentication** - Secure signup/login
3. **Scan History & Dashboard** - Save and track analyses
4. **PDF Processing** - Extract text from PDFs
